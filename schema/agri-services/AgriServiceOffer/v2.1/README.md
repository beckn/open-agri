# AgriServiceOffer — v2.1

**Schema Pack Version:** 2.1.0
**Released:** April 2026
**Status:** Initial release

## Notes

Extension of ServiceOffer for agriculture-specific services. Adds fields for specifying agricultural input brands and crop varieties.

### Container

- **Offer.offerAttributes**

### Key features

- Input brand or crop variety specification
- Extends all base ServiceOffer fields (quantity, booking terms, cancellation policy)
